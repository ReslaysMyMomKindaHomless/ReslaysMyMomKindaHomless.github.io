📤 Discord Webhook Sender

Envoie des fichiers sur Discord — option de connexion disponible

💡 Vous pouvez coller une URL de webhook ou vous connecter avec Discord (OAuth) pour afficher l'utilisateur connecté.
⚠️ Attention : l'envoi direct de fichiers vers l'API Discord depuis le navigateur peut être bloqué par CORS. Si vous rencontrez des erreurs, utilisez un relais serveur.
Pour que la connexion fonctionne, vous devez enregistrer exactement cette URL de redirection dans le dashboard Discord (ex: https://votre-site.github.io/index.html) et autoriser le scope identify.
